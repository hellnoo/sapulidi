SapuLidi
Sistem Analisis & Pengelolaan Limbah Cerdas
Aplikasi berbasis AI untuk mendukung penelitian disertasi S3 dalam bidang pengelolaan sampah. SapuLidi menggunakan teknologi computer vision untuk mendeteksi, mengklasifikasi, dan menganalisis jenis sampah secara otomatis.
Versi 1.0 · Prototipe Disertasi S3
🔍
Deteksi AI
Identifikasi jenis dan kategori sampah dari gambar menggunakan model AI berbasis Claude Vision
📚
Edukasi
Konten materi dan video edukasi terstruktur tentang pengelolaan sampah berbasis riset
📊
Dashboard
Visualisasi data dan statistik hasil deteksi untuk keperluan analisis penelitian
♻️
Klasifikasi
4 kategori: Organik, Anorganik, B3, dan Residu dengan penjelasan penanganan yang tepat
Teknologi
Claude AI Vision HTML5 / JavaScript Anthropic API Vercel Hosting
